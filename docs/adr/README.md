# Architecture Decision Records (ADRs)

This directory contains the Architecture Decision Records for the Livraison platform.

Each ADR captures a single significant decision: its context, the considered options, the chosen direction, and the consequences. ADRs are immutable once accepted; superseding decisions are added as new ADRs that reference the prior record.

## Status values

- **Proposed** — under discussion.
- **Accepted** — approved and in force.
- **Superseded by ADR-NNNN** — replaced by a later decision.
- **Deprecated** — no longer relevant; not yet superseded.
